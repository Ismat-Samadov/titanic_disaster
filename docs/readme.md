# Documentation Index

This folder contains project documentation split into focused guides.

## Contents

- `docs/usage.md` — how to run the pipeline and generate charts.
- `docs/technical_summary.md` — pipeline details, features, models, and outputs.

## Quick Start

1. Place `Titanic-Dataset.csv` in `data/`.
2. Run:

```bash
python scripts/generate_charts.py
```

Charts will be saved to `charts/`.
